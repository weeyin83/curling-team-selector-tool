Paste one player per line. Format: Name, primary, secondary, flags (secondary and flags are optional). Flags: flex, recent. Example: Sarah, third, skip, flex

Import players from an Excel workbook (.xlsx) or CSV (.csv) file. Each sheet is treated as a competition or league. The file must have Name and Position columns (extra columns are ignored). Positions are matched case-insensitively to skip / third / second / lead.

Leave the Position cell blank, or type any (also flex, flexible, anywhere, or *) for a player who can be placed in any position.

Select one sheet to add its players to the currently active tab, or select multiple sheets to create a separate competition tab for each sheet — so Trophy 1 and Trophy 2 stay as independent draws.

Everything is parsed locally in your browser — the file is never uploaded anywhere.
